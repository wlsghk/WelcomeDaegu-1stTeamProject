# 어서와, 대구는 처음이지?
<h3>📑 프로젝트 소개</h3>
<ul>대구 지역 브랜딩을 주제로 하여 MZ 세대를 타깃으로 설정하고 대구에 대한 이미지를 친근하고 특색 있게 느낄 수 있도록 디자인했습니다. 심리 테스트라는 단순한 소스를 통해 흥미를 유발하였고, 테스트별로 여행 레벨을 선정해 여행지를 추천했습니다.</ul>

<ul>2022. 11. 14. ~ 2022. 12. 9.</ul>
